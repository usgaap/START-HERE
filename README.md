# START-HERE
Start here to understand this repository.

US GAAP will "[go semantic](https://digitalfinancialreporting.blogspot.com/2024/02/fasb-and-us-gaap-go-semantic.html)" over the coming years.  This set of repositories will make this US GAAP metadata available to professional accountants and others.

The following information will be made available in machine interpretable form:

>[Disclosures](https://github.com/usgaap/disclosures) <br/>
>Working paper templates <br/>
>Line items <br/>
>Types and parts <br/>
>Conditions (a.k.a. assertions, restrictions, constraints) <br/>
>Key ratios (a.k.a. key performace indicators) <br/>
>Fundamental high level accounting concepts and their relations to other concepts <br/>
